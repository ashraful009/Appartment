# Frontend Directory Structure (client/src)

## Root Files
- `App.css`: Root styling for the application.
- `App.jsx`: Main application component, setting up routes and context providers.
- `index.css`: Global CSS file including Tailwind directives.
- `main.jsx`: Entry point for React rendering into the DOM.

## Assets
- `assets/react.svg`: React logo SVG.

## Context
- `context/AuthContext.jsx`: Global state context providing user authentication data and login/logout functions.

## Utils
- `utils/helpers.js`: Utility functions and generic helpers.

## Components - Admin
- `components/admin/analytics/AnalyticsShared.jsx`: Shared analytics charts/views.
- `components/admin/analytics/SellersShared.jsx`: Shared analytics components for seller statistics.
- `components/admin/dashboard/IdleLeadsWidget.jsx`: Widget showing leads that haven't been interacted with.
- `components/admin/dashboard/MonthlyTargetSetter.jsx`: UI for setting monthly targets for sellers.
- `components/admin/dashboard/StatCard.jsx`: Reusable card for displaying admin statistics.
- `components/admin/leads/LeadRow.jsx`: Table row displaying an individual lead's details.
- `components/admin/leads/PropertyCell.jsx`: Table cell displaying property details for a lead.
- `components/admin/sellers/PendingUserCard.jsx`: Card displaying a pending user registration.
- `components/admin/sellers/SellerProfileModal.jsx`: Modal to view and edit a seller's profile.
- `components/admin/sellers/SellerRow.jsx`: Table row displaying a seller's information.
- `components/admin/users/RoleToggles.jsx`: Component to toggle roles for users.
- `components/admin/users/Toggle.jsx`: Generic toggle switch UI.

## Components - Common
- `components/common/Avatar.jsx`: Reusable user avatar component.
- `components/common/FilterSidebar.jsx`: Sidebar component to filter properties or records.
- `components/common/HeroBanner.jsx`: Banner component typically used on the homepage.
- `components/common/Navbar.jsx`: Top navigation bar for the application.
- `components/common/PropertyCard.jsx`: Card component displaying a property's brief details.
- `components/common/PropertyGrid.jsx`: Grid layout to display multiple property cards.
- `components/common/TopBar.jsx`: Secondary top bar for specific layouts.

## Components - Customer 
- `components/customer/dashboard/CustomerStatCard.jsx`: Stat card for the customer dashboard.
- `components/customer/dashboard/NextActionBanner.jsx`: Banner prompting the customer for their next step.
- `components/customer/dashboard/WelcomeHeader.jsx`: Welcome message for logged-in customers.
- `components/customer/journey/InquiryCard.jsx`: Card showing a customer's specific inquiry.
- `components/customer/journey/InquiryStepper.jsx`: Step-by-step UI tracking an inquiry's progress.
- `components/customer/journey/InstallmentTable.jsx`: Table laying out payment installments.
- `components/customer/journey/PaymentPlanCard.jsx`: Display for a selected payment plan.
- `components/customer/journey/PaymentProgressBar.jsx`: Progress bar tracking payment completion.
- `components/customer/profile/AssignedAgentCard.jsx`: Card showing the customer's assigned agent.
- `components/customer/profile/CustomerProfileForm.jsx`: Form to update customer profile info.
- `components/customer/vault/DocumentCard.jsx`: Card displaying an uploaded document.
- `components/customer/vault/UploadDocumentModal.jsx`: Modal to upload a new document to the vault.

## Components - Seller
- `components/seller/claimed/ClaimedRow.jsx`: Table row for a claimed lead.
- `components/seller/claimed/ConversionToggle.jsx`: Toggle to mark a lead as converted.
- `components/seller/dashboard/TargetProgressBar.jsx`: Progress bar showing seller's monthly target achievement.
- `components/seller/leads/ActivityTimeline.jsx`: Timeline view of interactions with a lead.
- `components/seller/leads/AssignedRow.jsx`: Table row showing an assigned lead.
- `components/seller/leads/DelegateModal.jsx`: Modal for delegating a lead to another sub-seller.
- `components/seller/leads/ExpandedPanel.jsx`: Expanded view showing detailed lead actions.
- `components/seller/leads/FollowUpTaskCard.jsx`: Card showing a pending follow-up task.
- `components/seller/leads/InteractionForm.jsx`: Form to log a new interaction with a lead.
- `components/seller/leads/PipelineColumn.jsx`: Column for a Kanban-style pipeline view.
- `components/seller/profile/MentorCard.jsx`: Card displaying a mentor's info.
- `components/seller/profile/PerformanceStats.jsx`: Personal performance stats view for a seller.
- `components/seller/profile/ProfileForm.jsx`: Form for updating seller profile.
- `components/seller/profile/ReferralCodeWidget.jsx`: Widget showing a seller's referral code.
- `components/seller/team/BroadcastCard.jsx`: Card displaying broadcast messages.
- `components/seller/team/EarningsCard.jsx`: Card displaying earnings and commissions.
- `components/seller/team/SubSellerCard.jsx`: Card displaying details of a sub-seller in the team.
- `components/seller/team/SubSellerLeadRow.jsx`: Row displaying a lead assigned to a sub-seller.
- `components/seller/team/TimelineModal.jsx`: Modal showing a timeline of a sub-seller's activities.

## Pages
- `pages/admin/AddBuilding.jsx`: Admin page to add a new property/building.
- `pages/admin/AdminAnalytics.jsx`: Dashboard for administrative analytics.
- `pages/admin/AdminLayout.jsx`: Layout wrapper for admin pages.
- `pages/admin/AdminPendingLeads.jsx`: Page to review unassigned or pending leads.
- `pages/admin/AdminTimelineViewer.jsx`: Admin view of overall activity timelines.
- `pages/admin/BannerManagement.jsx`: Page to manage website banners.
- `pages/admin/Dashboard.jsx`: Main admin dashboard.
- `pages/admin/EditBuilding.jsx`: Page answering to edit an existing building.
- `pages/admin/ManageBuildings.jsx`: Table view to manage all buildings.
- `pages/admin/MasterAnalytics.jsx`: In-depth analytical reporting for administrators.
- `pages/admin/SellersAnalytics.jsx`: Analytics specifically focused on seller overall performance.
- `pages/admin/SellersPerformance.jsx`: Granular view of individual sellers' performances.
- `pages/admin/UserManagement.jsx`: Page to manage all registered users and roles.
- `pages/customer/CustomerDashboard.jsx`: Main dashboard for logged-in customers.
- `pages/customer/CustomerLayout.jsx`: Layout wrapper for customer pages.
- `pages/customer/CustomerProfilePage.jsx`: Page for the customer to view/edit their profile.
- `pages/customer/DocumentVaultPage.jsx`: Page allowing customers to access their uploaded documents.
- `pages/customer/JourneyPage.jsx`: Page showcasing a customer's specific buying/inquiry journey.
- `pages/public/CustomerProfile.jsx`: Public view of a customer profile.
- `pages/public/Home.jsx`: Main landing page of the application.
- `pages/public/Login.jsx`: Login page.
- `pages/public/PropertyDetails.jsx`: Public page showing full details of a single property.
- `pages/public/Register.jsx`: Registration page.
- `pages/seller/AssignedLeads.jsx`: View of leads currently assigned to the seller.
- `pages/seller/ClaimedUsers.jsx`: View of leads the seller has actively claimed.
- `pages/seller/MyTeam.jsx`: Dashboard for a seller to manage their team of sub-sellers.
- `pages/seller/PendingRequests.jsx`: Page to view leads pending action from the seller.
- `pages/seller/SellerDashboard.jsx`: Main seller dashboard overview.
- `pages/seller/SellerLayout.jsx`: Layout wrapper for seller portal pages.
- `pages/seller/SellerProfile.jsx`: Profile management tailored for seller users.
